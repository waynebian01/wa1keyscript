# 常用函数

## 获取玩家信息

```lua
Skippy.GetPlayerInfo()          -- 获取玩家所有信息
Skippy.GetPlayerAuras(auraName) -- 光环名称，如果有光环，返回光环，如果没有返回nil
```

## 法术完成状态

参数输入时间，返回布尔值，法术是否在最后指定时间完成：

```lua
Skippy.IsFinishedCasting(castTime)
Skippy.IsFinishedChanneling(channelTime)
```

## 获取指定 debuff 类型的单位

输入 `"Curse"`, `"Disease"`, `"Magic"`, `"Poison"`, `""`（`""` 是激怒效果），返回有指定 debuff 类型的单位：

```lua
Skippy.GetUnitHasDebuffWithdispelName(dispelName)
```

## 获取生命值最低的单位 (GetLowestUnit)

返回最低生命值的单位和当前生命值：

```lua
Skippy.GetLowestUnit()                                    -- 获取生命值最低的单位
Skippy.GetLowestUnitWithoutPlayer()                       -- 除玩家以外
Skippy.GetLowestUnitWithoutUnit(unitName)                 -- 除去单位以外，例：party1、raid2...
Skippy.GetLowestUnitWithRoles(role1, role2, role3)       -- 指定职责，如 "TANK", "HEALER", "DAMAGER"
Skippy.GetLowestUnitWithAura(auraName)                   -- 光环名称，如："恢复"
Skippy.GetLowestUnitWithPlayerAura(auraName)             -- 来自于玩家的光环名称
Skippy.GetLowestUnitWithoutPlayerAuras(auraName)         -- 来自于玩家的光环名称，返回没有该光环的单位
Skippy.GetLowestUnitWithPlayerAuraAndRole(auraName, role)       -- 来自于玩家的光环名称，指定职责
Skippy.GetLowestUnitWithoutPlayerAurasAndRole(auraName, role)  -- 来自于玩家的光环名称，指定职责，返回没有该光环的单位
Skippy.GetLowestUnitWithAnyAuras(auraTable)              -- 光环名称列表，如 { "恢复", "真言术：韧", "牺牲之手" }，返回有任意一个光环的单位
Skippy.GetLowestUnitWithAnyPlayerAuras(auraTable)        -- 来自于玩家的光环名称列表，如 { "恢复", "真言术：韧"}，返回有任意一个光环的单位
```

## 获取单位数量 (GetCount)

返回单位数量：

```lua
Skippy.GetCount(healthThreshold)                            -- 输入血量百分比阈值，生命值低于阈值的单位数量
Skippy.GetCountWithPlayerAura(healthThreshold, auraName)    -- 有来自于玩家的指定光环并生命值低于阈值的单位数量
Skippy.GetCountWithoutPlayerAura(healthThreshold, auraName) -- 没有来自于玩家的指定光环并生命值低于阈值的单位数量
Skippy.GetCountWithAnyAuras(healthThreshold, auraTable)     -- 有任意一个光环并生命值低于阈值的单位数量
Skippy.GetCountInSubGroup(healthThreshold)                  -- 指定小队中生命值低于阈值的单位数量
Skippy.GetCountInSubGroup(healthThreshold, subgroup)        -- 指定小队中生命值低于阈值的单位数量
```

## 获取单位 (GetUnit)

返回指定单位和单位当前生命值：

```lua
Skippy.GetUnitWithAura(auraName)                    -- 获取有指定光环的单位
Skippy.GetUnitWithoutAura(auraName)                 -- 获取没有指定光环的单位
Skippy.GetUnitWithAuraAndRole(auraName, role)       -- 获取指定职责有指定光环的单位
Skippy.GetUnitWithoutAuraAndRole(auraName, role)    -- 获取指定职责没有指定光环的单位
Skippy.GetUnitWithPlayerAura(auraName)              -- 获取有来自于玩家的指定光环的单位
Skippy.GetUnitWithoutPlayerAura(auraName)           -- 获取没有来自于玩家的指定光环的单位
Skippy.GetUnitWithPlayerAurasAndRole(auraName, role)    -- 获取指定职责有来自于玩家的指定光环的单位
Skippy.GetUnitWithoutPlayerAurasAndRole(auraName, role) -- 获取指定职责没有来自于玩家的指定光环的单位
```

## 获取单位和光环 (GetUnitAndAura)

返回指定单位和单位的光环信息：

```lua
Skippy.GetUnitAndAuraWithPlayerAura(auraName) -- 获取来自于玩家的光环信息
```

## 特殊函数

适用于团本欧米茄5号 boss，输入光环1和光环2，返回一个 table，同时拥有2个光环的单位信息：

```lua
Skippy.GetUnitWithAuraAndAuraCount(auraName, auraName2)
```
